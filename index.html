Low level C tutorial

geschreven door Luc de Wit

Waarom schrijven we C?

er word vaak gezecht dat C een "lage taal" is, dit wil zeggen dat het een laag abstractie level heeft. Dat betekend dan weer dat je "dicht bij de hardware" kan werken, en dus ook het geheugen van de computer direct kan beinvloeden.

Omdat het zo laag is, kan er ook veel optimizatie gedaan worden en is de taal veel sneller dan andere talen zoals python, C# of javascript. Wel is het zo dat je veel zelf moet doen wat in andere talen automatisch voor je gedaan wordt zoals: het bevrijden van gehuigen, en het reserveren van geheugen.


Hoe schrijven we C?

Om een C programma te kunnen runnen, moeten we een programma hebben dat onze C code kan lezen, veranderen in "machine taal" en deze machine taal dan kan optimizeren. zZ'n programma heet een compiler.

voor deze tutorial gebruiken we een online compiler die hier tevinden is. deze omgeving ziet er als volgt uit:


De standaard main functie

Dit is de code die je standaard krijgt als je replit laad: 

#include <stdio.h>

int main(void) {
	printf("Hello World\n");
	return 0;
}
Laten we deze code lijn voor lijn gaan analyzeren. 


#include <stdio.h>
Deze lijn kopieert alle code uit een file genaamt "stdio.h" en plakt het waar deze lijn code staat. In deze file zitten een hoop functies die je kunt gebruiken om text te kunnen printen in de console (de console is de rechter helft van het scherm in repl.it, hier word je code in ge runned). 


int main(void) {
Deze lijn kondigd aan dat er een functie gemaakt moet worden die een integer returned (in C, een integer is een geheel nummer dat 4 bytes geheugen in beslag neemt), de functie heet main, en hij heeft geen (void betekend geen) argrumenten nodig.

De main functie is een speciale functie, hij word namenlijk automatisch gecalled wanneer je het programma start, deze functie heet ook wel de "entrypoint" van het programma. 


printf("Hello World\n");
Deze lijn zit in de main functie, en is dus het eerste wat er gebeurt als je het programma start, het called een functie genaamt printf, en geeft de string "Hello World\n" (\n is een place holder voor het enter symbool). Printf is een functie uit de stdio.h "header" en neemt de string die als eerste argrument is gegeven (in dit geval Hello World) en print in de console. 


return 0;
dit returned het integer 0, en zal dus de main functie verlaten, dit betekend dus dat het programma over is, en 0 betekend dat er geen error is opgetreden.

Variabelen

Variabelen in C lijken erg op de variabelen van C++ en C#, variabelen hebben de volgende syntax:
- een datatype (om aan te geven wat voor soort type je wilt opslaan (dit is nodig voor het reserveren van het geheugen)
- een naam, zodat je later in je code kan terug refereren naar het variabele
- een = teken
- een waarde (dit kan een nummer zijn, maar ook een string of zelfs een ander variabele)
- een ; om de lijn af te sluiten

Hier is een voorbeeld hoe je een variabele maakt: 

int main() {
	int funnyNumber = -69420;
	unsigned int otherFunnyNumber = 69;
	float anotherNumber = 656.234;
	char tinyNumber = 42;
	return 0;
}
Een int is een integer, dat zal 4 bytes aan geheugen reserveren

Unsigned is een type modifier, die ervoor zorgt dat er geen negatieve getallen kunnen worden opgeslagen

Float is een decimaal getal, en heeft ook 4 bytes nodig

Char is een klein getal van 0 tot 255, en heeft 1 byte nodig

De waarde van char, kan ook gezet worden met de volgende syntax: 

int main() {
	char tinyNumber = 'A';
	return 0;
}
In dit geval zal het symbool 'A' omgezet worden naar zijn ascii waarde (dat is in dit geval 65), en dat zal dan worden opgeslagen in het variabele.

PAS OP: gebruik single qoutes (') en niet double quotes (") om aan te geven dat het een character is, en niet een string

Een heel belangrijk datatype waar we het nog niet over hebben gehad, is een string, in princiepe heeft C geen strings, maar je kunt wel een array van characters gebruiken, dit doe je als volgt: 

int main() {
	char naam[] = "henk";
	return 0;
}

In C kun je ook variabelen printen, dit kun je doen met de printf functie. je kunt in de string een placeholder zetten, en dan een variabele invullen in die placeholder, bijvoorbeeld:

"mijn naam is %s", dit is een string, met een place holder (%s, dat betekend een placeholder voor een string), voor een naam. voor een integer kun je %d gebruiken, %c is voor een char en voor een float kun je %f gebruiken. Je kunt met printf variabelen printen op deze manier: 

int main() {
	int aantal = 5;
	char objecten[] = "kroketten";
	printf("ik heb een %s aantal %s", aantal, objecten);
	return 0;
}
